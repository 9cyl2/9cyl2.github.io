Session 14: 参数估计（点估计）

Session 14: 什么是参数估计？

引入： 我们知道全校男生的身高服从正态分布 $N(\mu, \sigma^2)$，但我们不知道具体的均值 $\mu$ 和方差 $\sigma^2$ 到底是多少。这两个未知的常数就叫未知参数。

点估计 (Point Estimation) 的核心任务
从总体中抽取一个样本 $X_1, X_2, \dots, X_n$，利用构造的公式算出一个具体的数值，直接作为未知参数 $\theta$ 的估计值。

体育大白话： 就像球探去考察一个新球员，看他打了 5 场比赛（抽取样本），然后向老板汇报：“我估计他真实的场均得分能力是 22 分”。这个“22分”就是点估计值！

Session 14: 绝招一 —— 矩估计法 (Method of Moments)

核心思想（简单粗暴）： 用“样本”的特征去强行代替“总体”的特征！
即：“样本均值”代替“总体期望”，“样本方差”代替“总体方差”。

矩估计法的操作步骤
  1. 列方程： 令总体的期望 $E(X)$ 等于样本均值 $\bar{X}$。（如果有两个未知参数，就再令总体的二阶矩 $E(X^2)$ 等于样本的平方均值 $\frac{1}{n}\sum X_i^2$）。
  2. 解方程： 把未知参数当成未知数，从上述等式中解出来。解出来的表达式戴上“帽子 ($\hat{}$)”即为矩估计量。

Session 14: "Sports Teacher” 专属例题 (矩估计)

例题： 体育课上测某项反应速度，假设学生的反应时间 $X$ 服从均匀分布 $U(0, \theta)$，其中最高延迟 $\theta$ 未知。现抽取 5 名学生，成绩为：$0.2, 0.4, 0.3, 0.5, 0.6$（秒）。
求 $\theta$ 的矩估计值。

解析：

  • 总体 $X \sim U(0, \theta)$，根据上节课背诵的表格，其期望 $E(X) = \frac{0 + \theta}{2} = \frac{\theta}{2}$。
  • 令总体期望等于样本均值：$E(X) = \bar{X} \implies \frac{\theta}{2} = \bar{X}$。
  • 解出参数 $\theta$：$\hat{\theta} = 2\bar{X}$。这就是矩估计量的公式。
  • 代入具体的样本数据：$\bar{x} = \frac{0.2+0.4+0.3+0.5+0.6}{5} = \frac{2.0}{5} = 0.4$。
  • 结论： 参数 $\theta$ 的矩估计值为 $\hat{\theta} = 2 \times 0.4 = 0.8 \text{ 秒}$。

Session 14: 绝招二 —— 极大似然估计 (MLE)

核心思想（存在即合理）： 既然这组特定的样本在几十万种可能性中被我抽中了，说明这组样本发生的概率（似然度）必定是极其大的！我们就去寻找那个能让概率最大化的参数。

极大似然估计 (MLE) 期末必考四步曲
  1. 写出似然函数 $L(\theta)$： 把所有独立样本的概率（或密度）连乘起来：
    $L(\theta) = \prod_{i=1}^n f(x_i; \theta)$
  2. 取对数（极其关键！）： 对 $L(\theta)$ 取自然对数 $\ln L(\theta)$，把连乘变成连加，大大简化求导！
  3. 求导并令其为 0： 令 $\frac{d \ln L(\theta)}{d\theta} = 0$，解出 $\theta$。
  4. 带上帽子： 解出的 $\hat{\theta}$ 即为极大似然估计量。

Session 14: 极大似然估计的几何直观

通过求导寻找 $\ln L(\theta)$ 曲线的最高点，该最高点对应的横坐标就是我们苦苦寻找的最优参数 $\hat{\theta}_{MLE}$。

Session 14: "Sports Teacher” 专属例题 (极大似然)

例题： 考察某球员的三分球命中率 $p$（未知）。让该球员投篮 $n$ 次，发现命中了 $k$ 次。请用极大似然法估计他的真实命中率 $p$。

解析：

  • 1. 写似然函数： 投中 $k$ 次，没中 $n-k$ 次，其发生的总概率（即似然函数）为：
    $L(p) = C_n^k p^k (1-p)^{n-k}$
  • 2. 取对数： （常数 $C_n^k$ 求导时会变为0，可不管）
    $\ln L(p) = \ln(C_n^k) + k \ln p + (n-k) \ln(1-p)$
  • 3. 求导等于 0： 对参数 $p$ 求导：
    $\frac{d \ln L(p)}{dp} = \frac{k}{p} - \frac{n-k}{1-p} = 0$
  • 4. 解方程： 交叉相乘得 $k(1-p) = p(n-k) \implies k - kp = np - kp \implies \hat{p} = \textcolor{#e74c3c}{\frac{k}{n}}$。
  • 结论： 命中率的最大似然估计就是他的样本命中频率！存在即合理，非常符合直觉！

Session 14: 评判估计量的好坏 —— 无偏性

同一个参数，用不同方法可能算出不同的估计式。这就需要评判标准，最核心的就是无偏性 (Unbiasedness)：即估计量 $\hat{\theta}$ 的数学期望刚好等于真实参数 $\theta$。

无偏估计
(打得散，但整体中心对准靶心)

有偏估计
(打得密，但准星存在系统偏差)

随堂小测：概念与计算考核

选择题挑战

1. 在矩估计法中，如果是单参数的估计，我们通常令（ ）等于样本均值 $\bar{X}$。

  1. 总体的方差 $D(X)$
  2. 总体的极大似然函数 $L(\theta)$
  3. 总体的数学期望 $E(X)$
  4. 样本的方差 $S^2$
👉 答案：C
解析：矩估计的核心思想就是“用样本矩代替总体矩”。一阶样本矩就是样本均值 $\bar{X}$，对应的一阶总体矩就是总体的数学期望 $E(X)$。

2. 设 $\hat{\theta}$ 是总体未知参数 $\theta$ 的一个估计量。若 $\hat{\theta}$ 是无偏估计，则必须满足（ ）。

  1. $D(\hat{\theta}) = \theta$
  2. $E(\hat{\theta}) = \theta$
  3. $\hat{\theta} = \theta$
  4. $\lim_{n \to \infty} P(|\hat{\theta} - \theta| \lt \varepsilon) = 1$
👉 答案：B
解析：B项是无偏性的严格数学定义，即估计量的数学期望等于被估计的真实参数。

概念与计算考核

考核 1：概念简答 为什么在进行“极大似然估计 (MLE)”计算时，我们通常要在求导之前，先对似然函数 $L(\theta)$ 取自然对数 $\ln$ 呢？

👉 答案： 因为似然函数 $L(\theta)$ 是由 n 个样本的概率连乘得到的，直接对连乘积求导极其复杂且容易出错。取对数后，利用 $\ln(a \cdot b) = \ln(a) + \ln(b)$ 的性质，可以将连乘转化为连加，极大地简化了后续的求导运算。由于 $\ln(x)$ 是单调递增函数，使 $\ln L(\theta)$ 最大的参数 $\theta$，也必定能使原似然函数 $L(\theta)$ 最大。

考核 2：计算挑战 设总体 $X$ 服从参数为 $\lambda$ 的泊松分布 $P(\lambda)$。现取得样本 $x_1, x_2, \dots, x_n$。已知泊松分布的期望 $E(X) = \lambda$。请用矩估计法求参数 $\lambda$ 的矩估计值。

👉 解析：

  • 第一步（列方程）：写出总体的一阶矩，即期望 $E(X) = \lambda$。
  • 第二步（代入样本）：令总体期望等于样本均值，即 $E(X) = \bar{X}$。
  • 第三步（解方程）：代入得 $\lambda = \bar{X}$。
  • 第四步（戴帽子）：所以 $\lambda$ 的矩估计量为 $\hat{\lambda} = \bar{X}$。
