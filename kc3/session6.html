Session 6: 连续型随机变量及其分布

Session 6: 连续型随机变量与概率密度函数

引入： 离散型变量可以“一个一个数出来”，但连续型变量（如百米冲刺时间）是连续不断的。

概率密度函数 (Probability Density Function, PDF)
若存在非负可积函数 $f(x)$，使得对于任意 $a < b$，都有： $$ P(a < X < b) = \int_{a}^{b} f(x) dx $$ 则称 $X$ 为连续型随机变量，$f(x)$ 为其概率密度函数。

Session 6: 概率密度的核心性质 (极其重要)

连续型随机变量的概率，本质上就是求密度函数曲线下的面积！

  • 1. 非负性： $f(x) \ge 0$ （密度不能是负的）。
  • 2. 规范性： $\int_{-\infty}^{+\infty} f(x) dx = 1$ （曲线下总面积为1，即总概率为1）。
  • 3. 单点概率为零： 对于确定的数值 $a$，$P(X=a) = \int_{a}^{a} f(x) dx = 0$。
易错点防坑
对于连续型随机变量，“概率为0”不代表“不可能发生”！而且计算区间概率时，边界带不带等号都一样： $$ P(a \le X \le b) = P(a < X < b) $$

Session 6: 常见连续分布一 —— 均匀分布

物理意义： 变量在某个区间内取任何值的可能性是“均等”的。

均匀分布 $U(a,b)$
若 $X$ 的密度函数在区间 $(a,b)$ 上是常数，在其他地方为 0，即： $$ f(x) = \begin{cases} \frac{1}{b-a}, & a < x < b \\ 0, & \text{其他} \end{cases} $$

Session 6: 常见连续分布二 —— 指数分布

物理意义： 常用于描述“等待时间”或“寿命”。比如等待空出篮球场的时间、体测计时器的寿命。

指数分布 $E(\lambda)$
若 $X$ 的密度函数为： $$ f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \le 0 \end{cases} \quad (\lambda > 0) $$ 则称 $X$ 服从参数为 $\lambda$ 的指数分布。

神奇特性（无记忆性）： $P(X > s+t \mid X > s) = P(X > t)$。
（你等校车等了 10 分钟没来，接下去还要等的时间概率，和刚开始等是一模一样的！）

Session 6: 常见连续分布三 —— 正态分布

自然界中受大量微小独立因素叠加影响的变量（如身高、体测成绩）都服从正态分布。

正态分布 (高斯分布) $N(\mu, \sigma^2)$
其密度函数为标志性的“钟形曲线”： $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} $$
  • $\mu$ 决定了中心位置；$\sigma$ 决定了高矮胖瘦（越小越陡峭）。

Session 6: 标准正态分布与查表法

当 $\mu = 0, \sigma = 1$ 时，我们称之为标准正态分布，记为 $Z \sim N(0,1)$。可通过查表直接得到结果。

正态分布的标准化转换 (极其重要！)
任何一般正态分布 $X \sim N(\mu, \sigma^2)$ 都可以通过“减去均值，除以标准差”的方法，转化为标准正态分布： $$ Z = \frac{X - \mu}{\sigma} \sim N(0, 1) $$

对称性公式： $\Phi(-z) = 1 - \Phi(z)$。

Session 6: "Sports Teacher” 专属例题 (正态分布应用)

例题： 本学期全校大二男生的 1000 米体测成绩 $X$（单位：秒）近似服从正态分布 $N(230, 20^2)$。（即平均 230 秒，标准差 20 秒）。及格线为 260 秒。

求：随机抽查一名男生，他体测及格（即成绩 $X \le 260$）的概率。

解析：

  • 目标是求 $P(X \le 260)$。由于 $X \sim N(230, 20^2)$，我们需要标准化。
  • 令 $Z = \frac{X - 230}{20}$，则 $Z \sim N(0,1)$。
  • 不等式两边同时做标准化处理：
    $$ P(X \le 260) = P\left( \frac{X - 230}{20} \le \frac{260 - 230}{20} \right) = P(Z \le 1.5) $$
  • 查表得到：$\Phi(1.5) = \textcolor{#e74c3c}{0.9332}$。结论：及格率约为 93.32%。

随堂小测：概念与计算考核

选择题挑战

1. 设连续型随机变量 $X$ 的概率密度函数为 $f(x)$，则下列关于 $f(x)$ 的描述必定正确的是（ ）。

  1. $f(x) \le 1$ 对所有 $x$ 成立
  2. $\int_{-\infty}^{+\infty} f(x) dx = 1$
  3. $f(x)$ 表示 $X$ 在 $x$ 处取值的概率
  4. $f(x)$ 必须是连续函数
👉 答案：B
解析：B 是密度函数的规范性定义。A错，只要总面积为1，高度可以大于1；C错，$f(x)$ 并非概率本身（连续变量单点概率为0）；D错，均匀分布的密度函数就存在断点。

2. 已知某学生练习投掷实心球的落点距离 $X$ 是一个连续型随机变量，那么 $P(X = 10.5\text{米})$ 的概率等于（ ）。

  1. 1
  2. $0.5$
  3. 不能确定
  4. $0$
👉 答案：D
解析：这是连续型随机变量的核心性质之一，单点概率必定为 0。

概念与计算考核

考核 1： 若随机变量 $X \sim N(5, 4)$，请问该分布的均值 $\mu$ 和标准差 $\sigma$ 分别是多少？转化公式是？

👉 答案： 均值 $\mu = 5$，方差为 4，所以标准差 $\sigma = 2$（要开根号！）。转化公式为：$Z = \frac{X - 5}{2}$。

考核 2： 已知随机变量 $X$ 服从区间 $(0, 10)$ 上的均匀分布，写出它的概率密度函数 $f(x)$，并计算 $P(2 < X < 5)$ 的值。

👉 解析：

  • 因为区间长度 $10 - 0 = 10$，所以在 $(0,10)$ 上，$f(x) = \frac{1}{10} = 0.1$。其他地方 $f(x)=0$。
  • 求区间概率，即求矩形面积：
    $$ P(2 < X < 5) = \int_{2}^{5} 0.1 dx = 0.1 \times (5 - 2) = 0.3 $$
