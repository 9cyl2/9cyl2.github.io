Session 9: 二维随机变量的函数的分布

Session 9: 为什么研究二维变量的函数？

引入： 在体育竞技中，很多最终成绩是多个指标的“组合”。

  • 加法模型： 4×100米接力赛，总成绩 $Z$ 是第一棒 $X$ 和第二棒 $Y$ 的时间之和，即 $Z = X + Y$。
  • 最值模型： 跳远比赛允许跳两次（成绩为 $X$ 和 $Y$），最终成绩取最好的一次，即 $Z = \max(X, Y)$。
核心目标
已知 $(X,Y)$ 的联合分布，求 $Z = g(X,Y)$ 的分布。

Session 9: 离散型 —— 简单粗暴的“穷举法”

对于离散型随机变量，求 $Z = g(X,Y)$ 的分布律，方法和一维完全一样！
口诀：“算出所有可能的 Z，把 Z 相同的概率加起来”。

例题： 设 $(X,Y)$ 的联合分布律如下，求 $Z = X + Y$ 的分布律。

$X \setminus Y$$0$$1$
$0$$0.2$$0.3$
$1$$0.1$$0.4$

解析合并过程：

  • $X,Y$ 组合有：$(0,0), (0,1), (1,0), (1,1)$。对应的 $Z$ 分别为：$0, 1, 1, 2$。
  • 合并同类项（$Z=1$ 包含两种情况）：
    $P(Z=0) = 0.2$
    $P(Z=1) = P(X=0,Y=1) + P(X=1,Y=0) = 0.3 + 0.1 = \textcolor{#e74c3c}{0.4}$
    $P(Z=2) = 0.4$

Session 9: 连续型 —— 两个变量的和 Z = X+Y

问题： 设连续型变量 $(X,Y)$ 具有联合密度 $f(x,y)$，求 $Z = X+Y$ 的密度函数 $f_Z(z)$。

利用分布函数法 $F_Z(z) = P(X+Y \le z) = \iint_{x+y \le z} f(x,y) dx dy$ 并在求导后，得到了著名的卷积公式 (Convolution)：

卷积公式
$$ f_Z(z) = \int_{-\infty}^{+\infty} f(x, z-x) dx \quad \text{或} \quad f_Z(z) = \int_{-\infty}^{+\infty} f(z-y, y) dy $$

Session 9: "Sports Teacher” 专属例题 (加法模型)

例题 (独立正态的叠加)： 某两名田径队员进行接力赛，第一棒用时 $X \sim N(12, 1^2)$，第二棒用时 $Y \sim N(11, 2^2)$，且两人发挥相互独立。求他们总用时 $Z = X+Y$ 的分布。

解析： 这里给大家介绍一个超级大招（期末可直接用）——正态变量的“再生性”。

正态分布的线性可加性
如果 $X$ 和 $Y$ 相互独立，且都服从正态分布，那么它们的和 $Z = X+Y$ 依然服从正态分布！
  • 新均值： $\mu_Z = \mu_X + \mu_Y$
  • 新方差： $\sigma^2_Z = \sigma^2_X + \sigma^2_Y$

代入数据：
均值 $\mu_Z = 12 + 11 = 23$。
方差 $\sigma^2_Z = 1^2 + 2^2 = 1 + 4 = 5$。
结论： 两人接力总用时 $Z \sim N(23, 5)$。

Session 9: 最大值与最小值分布 (极值分布)

在实际情况中，我们常关心一个系统的瓶颈（木桶效应、最小值）或者最佳表现（最好成绩、最大值）。设 X 和 Y 是相互独立的随机变量，分布函数分别为 $F_X(x)$ 和 $F_Y(y)$。

1. 最大值分布 $M = \max(X,Y)$
$M \le z$ 意味着 X 和 Y 同时小于等于 z。
$$ F_{\max}(z) = P(\max(X,Y) \le z) = P(X \le z, Y \le z) $$ 因为独立，所以：$F_{\max}(z) = F_X(z) \cdot F_Y(z)$
2. 最小值分布 $N = \min(X,Y)$ (正难则反！)
$N \gt z$ 意味着 X 和 Y 同时大于 z。
$$ P(N \gt z) = P(X \gt z, Y \gt z) = P(X \gt z)P(Y \gt z) $$ 所以：$F_{\min}(z) = 1 - [1-F_X(z)][1-F_Y(z)]$

Session 9: "Sports Teacher” 专属例题 (极值模型)

例题： 在跳远测试中，某学生有两次试跳机会，两次成绩 $X, Y$ 相互独立，且都服从区间 $[4, 6]$ 上的均匀分布。最终成绩取两次中的最大值 $Z = \max(X,Y)$。求 Z 的分布函数。

解析：

  • 首先，单次跳远 $X \sim U(4,6)$ 的分布函数为：
    $F_X(z) = 0$ (当 $z \lt 4$)； $F_X(z) = \frac{z-4}{2}$ (当 $4 \le z \lt 6$)； $F_X(z) = 1$ (当 $z \ge 6$)。
  • 既然 X 和 Y 同分布且独立，根据最大值公式：
    $F_Z(z) = F_{\max}(z) = F_X(z) \cdot F_Y(z) = [F_X(z)]^2$
  • 最终成绩的分布函数直接平方即为：
    $F_Z(z) = 0$ (当 $z \lt 4$)；
    $F_Z(z) = \frac{(z-4)^2}{4}$ (当 $4 \le z \lt 6$)；
    $F_Z(z) = 1$ (当 $z \ge 6$)。

随堂小测：概念与计算考核

选择题挑战

1. 已知独立随机变量 $X \sim N(0, 1)$ 和 $Y \sim N(0, 1)$，则 $Z = X + Y$ 服从（ ）。

  1. $N(0, 1)$
  2. $N(0, 2)$
  3. $N(0, \sqrt{2})$
  4. 无法确定
👉 答案：B
解析：根据独立正态分布的可加性，均值相加 $0+0=0$，方差相加 $1+1=2$。所以 $Z \sim N(0,2)$。

2. 设系统由两个相同的部件串联而成（只要有一个坏，系统就崩溃）。若部件寿命相互独立且分布函数均为 $F(x)$，则系统寿命 $Z$ 的分布函数为（ ）。

  1. $F(x) \cdot F(x)$
  2. $1 - [1-F(x)]^2$
  3. $2F(x)$
  4. $F(x)$
👉 答案：B
解析：串联系统的寿命实际上等于两个部件寿命的“最小值”（短板效应）。因此 $Z = \min(X,Y)$，套用公式 $F_{\min}(z) = 1 - [1-F_X(z)][1-F_Y(z)]$ 即得 B。

概念与计算考核

考核 1：简述 为什么在求解最大值分布 $M = \max(X,Y) \le z$ 时，可以直接转化为 $X \le z$ 且 $Y \le z$？

👉 答案： 因为如果一组数中的“最大值”都已经小于或等于 $z$ 了，那么这组数里的“所有其他数”必定也都小于或等于 $z$。这是一个极其重要的等价事件转化逻辑。

考核 2：计算 设离散型变量 $X$ 取 0, 1，概率各为 0.5；$Y$ 取 0, 1，概率也各为 0.5。且 $X, Y$ 相互独立。求 $Z = \max(X,Y)$ 为 0 的概率。

👉 解析：

  • 要求 $Z = \max(X,Y) = 0$。
  • 因为 $X, Y$ 只能取 0 或 1，所以最大值为 0 的唯一情况就是：$X=0$ 且 $Y=0$ 同时发生。
  • 因为相互独立，所以 $P(\max(X,Y)=0) = P(X=0, Y=0) = P(X=0) \times P(Y=0) = 0.5 \times 0.5 = 0.25$。
