Session 10: 随机变量的数字特征

期望与方差

Session 10: 为什么需要数字特征？

引入： 要评价一个篮球运动员的水平，把他的每场得分分布律全拿出来看太复杂了。教练往往只问两个问题：

  • 他平均每场能拿多少分？
  • 他发挥得稳定吗？（是神鬼莫测，还是稳如老狗？）
数字特征的核心意义
分布律或密度函数能提供完整信息，但往往难以获取且过于繁琐。
数字特征（如数学期望、方差）能用极简的常数高度概括随机变量的最重要性质！

Session 10: 数学期望 (Mathematical Expectation)

数学期望，简称期望或均值，反映了随机变量取值的“重心”或“加权平均水平”。

期望的计算公式
  • 离散型： “取值 $\times$ 概率，全部加起来”。 $$ E(X) = \sum_{k=1}^{\infty} x_k p_k $$ （要求级数绝对收敛，否则期望不存在）
  • 连续型： “取值 $\times$ 密度，在全空间积分”。 $$ E(X) = \int_{-\infty}^{+\infty} x f(x) dx $$

Session 10: 期望的物理意义 —— 重心

数学期望 $E(X)$ 在物理学上，完美对应了一根质量不均匀长杆的重心（质心）！

期望的四大性质 (线性性质，必考！)
  1. $E(C) = C$ (常数的期望还是常数)
  2. $E(aX) = a E(X)$ (常数系数可直接提出来)
  3. $E(X + Y) = E(X) + E(Y)$ (无条件成立，不需要独立！)
  4. 推论：$E(aX + bY + c) = a E(X) + b E(Y) + c$

Session 10: "Sports Teacher” 专属例题 (数学期望)

例题： 某篮球队核心后卫在比赛中的单场得分 X 的分布律如下表：

$X$ (得分)15202530
$p_k$ (概率)$0.1$$0.4$$0.3$$0.2$

求：(1) 他的场均期望得分 $E(X)$。
(2) 若球队设奖金，每得一分奖 1000 元，底薪 5000 元，求他单场奖金 Y 的期望 $E(Y)$。

解析：

  • (1) 根据定义 $E(X) = \sum x_k p_k$：
    $E(X) = 15 \times 0.1 + 20 \times 0.4 + 25 \times 0.3 + 30 \times 0.2 = 1.5 + 8 + 7.5 + 6 = \textcolor{#e74c3c}{23 \text{ 分}}$。
  • (2) 奖金函数关系为 $Y = 1000X + 5000$。利用期望的线性性质：
    $E(Y) = E(1000X + 5000) = 1000E(X) + 5000 = 1000 \times 23 + 5000 = \textcolor{#2980b9}{28000 \text{ 元}}$。

Session 10: 方差 (Variance) 与标准差

知道了平均水平，我们还需要知道数据波动大小（偏离均值的程度），这就引入了方差。

方差的定义与计算公式
定义式： $D(X) = E \{ [X - E(X)]^2 \}$ （即“偏差平方”的平均值）。
计算公式（极度重要！）： $D(X) = E(X^2) - [E(X)]^2$

由于方差被平方过，为了恢复原数据的单位，我们定义标准差： $$ \sigma(X) = \sqrt{D(X)} $$

方差的性质 (注意与期望的区别！)
  • $D(C) = 0$ (常数不波动，方差为0)
  • $D(aX) = a^2 D(X)$ (系数提出来必须平方！)
  • $D(X + C) = D(X)$ (加减常数平移，不改变波动程度)
  • 若 X 和 Y 相互独立，才有 $D(X \pm Y) = D(X) + D(Y)$。

Session 10: "Sports Teacher” 专属例题 (方差对比)

例题： 甲、乙两名射击选手的单次环数分布如下。已知两人平均环数相同 $E(X)=E(Y)=9$。请利用方差判断，教练应派谁去比赛？

甲选手

$X$8910
$p_k$0.20.60.2

乙选手

$Y$8910
$p_k$0.40.20.4

解析： 平均分相同，比稳定性（方差越小越稳定）。

  • 算平方的期望 $E(X^2)$：
    $E(X^2) = 64(0.2) + 81(0.6) + 100(0.2) = 12.8 + 48.6 + 20 = 81.4$
    $E(Y^2) = 64(0.4) + 81(0.2) + 100(0.4) = 25.6 + 16.2 + 40 = 81.8$
  • 算方差 $D(X) = E(X^2) - [E(X)]^2$：
    $D(X) = 81.4 - 9^2 = 81.4 - 81 = \textcolor{#2980b9}{0.4}$
    $D(Y) = 81.8 - 9^2 = 81.8 - 81 = \textcolor{#e74c3c}{0.8}$
  • 结论： 甲的方差较小 ($0.4 \lt 0.8$)，甲发挥更稳定，派甲参赛！

随堂小测：概念与计算考核

选择题挑战

1. 已知随机变量 $X$ 的方差为 $D(X) = 4$，则 $D(-2X + 3)$ 的值为（ ）。

  1. -5
  2. 16
  3. 11
  4. -8
👉 答案：B
解析：考察方差的性质。常数平移（+3）不影响方差，即 $D(-2X+3) = D(-2X)$。提出系数要平方：$D(-2X) = (-2)^2 D(X) = 4 \times 4 = 16$。

2. 设随机变量 $X$ 服从泊松分布 $P(4)$，则 $E(X)$ 与 $D(X)$ 的值分别是（ ）。

  1. $E(X)=4, D(X)=2$
  2. $E(X)=4, D(X)=16$
  3. $E(X)=4, D(X)=4$
  4. $E(X)=16, D(X)=4$
👉 答案：C
解析：泊松分布的超级特性：它的期望和方差不仅存在，而且数值完全相等，都等于参数 $\lambda$。本题 $\lambda=4$，所以都等于 4。

概念与计算考核

考核 1：简答题 在体育数据分析中，“数学期望”和“方差”分别代表了运动员成绩的什么现实物理意义？

👉 答案： 数学期望代表了运动员的平均实力水平（长期的、整体的发挥中心）；方差代表了运动员表现的稳定性（波动程度），方差越小，发挥越稳定。

考核 2：计算题 抛掷一枚均匀的正六面体骰子，令 $X$ 表示出现的点数。已知 $E(X) = 3.5$。请计算 $E(X^2)$ 和 $D(X)$。

👉 解析：

  • $X$ 的取值为 $1,2,3,4,5,6$，概率均为 $\frac{1}{6}$。
  • 求 $E(X^2)$：$E(X^2) = 1^2(\frac{1}{6}) + 2^2(\frac{1}{6}) + ... + 6^2(\frac{1}{6}) = \frac{1+4+9+16+25+36}{6} = \frac{91}{6} \approx 15.17$。
  • 求 $D(X)$：利用公式 $D(X) = E(X^2) - (EX)^2 = \frac{91}{6} - (3.5)^2 = \frac{91}{6} - \frac{49}{4} = \frac{35}{12} \approx 2.92$。
