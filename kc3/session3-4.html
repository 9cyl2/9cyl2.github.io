Session 3: 条件概率与贝叶斯公式

Session 3: 条件概率 (缩小了的样本空间)

引入： 在已知某事件已经发生的“情报”下，另一个事件发生的概率会改变吗？

条件概率的定义
设 A, B 是两个事件，且 $P(A) > 0$，则在事件 A 已经发生的条件下，事件 B 发生的条件概率定义为： $$ P(B|A) = \frac{P(AB)}{P(A)} $$

直观几何意义： 样本空间从原来的整体 S 缩小到了 A。

Session 3: 乘法公式 (由条件概率推导)

将条件概率的分母乘过去，我们就得到了极其强大的乘法公式：

  • 两个事件： $$ P(AB) = P(A)P(B|A) \quad (\text{若 } P(A) > 0) $$
  • 三个事件： $$ P(ABC) = P(A)P(B|A)P(C|AB) $$
  • 本质理解： 事情是一步一步发生的，后面的概率必须建立在前面已经发生的基础上。

Session 3: 样本空间的划分 (完备事件组)

在引入全概率公式之前，我们需要把复杂的样本空间“切蛋糕”。

划分的定义
设 $A_1, A_2, \dots, A_n$ 为一组事件，若满足：
  1. 互不相容： $A_i A_j = \emptyset \quad (i \neq j)$ （即两两不重叠）
  2. 不重不漏： $A_1 \cup A_2 \cup \dots \cup A_n = S$ （拼起来是整个样本空间）
则称 $A_1, A_2, \dots, A_n$ 为样本空间 $S$ 的一个划分。

Session 3: 全概率公式 (由因求果)

当一个复杂事件 B 的发生受多种互斥原因 $A_i$ 影响时，我们可以把概率“分块求和”。

全概率公式
设 $A_1, A_2, \dots, A_n$ 是样本空间的一个划分，且 $P(A_i) > 0$。则对任一事件 B，有： $$ P(B) = \sum_{i=1}^n P(A_i)P(B|A_i) $$

解题口诀： “权重 $\times$ 条件概率，然后全部加起来”。
常用工具：概率树状图。

Session 3: 书本经典例题 (全概率与次品率)

例题： 甲、乙、丙三台机床生产同一种零件，产量占比分别为 30%, 20%, 50%，次品率分别为 2%, 3%, 1%。现从中任取一件，求它是次品的概率。

解析：

  • 设 $A_1, A_2, A_3$ 为零件由甲、乙、丙生产（原因），B 为抽到次品（结果）。
  • 提取已知先验概率：$P(A_1)=0.3, P(A_2)=0.2, P(A_3)=0.5$。
  • 提取已知条件概率：$P(B|A_1)=0.02, P(B|A_2)=0.03, P(B|A_3)=0.01$。
  • 代入全概率公式： $$ P(B) = P(A_1)P(B|A_1) + P(A_2)P(B|A_2) + P(A_3)P(B|A_3) $$ $$ P(B) = 0.3 \times 0.02 + 0.2 \times 0.03 + 0.5 \times 0.01 = \textcolor{#e74c3c}{0.017} $$

Session 3: 贝叶斯公式 (执果索因)

灵魂拷问： 如果上题中，我查出这件零件已经是次品了，请问它是甲机床生产的概率有多大？（老板要追责了！）

贝叶斯公式 (Bayes' Theorem)
$$ P(A_i|B) = \frac{P(A_i)P(B|A_i)}{\sum_{j=1}^n P(A_j)P(B|A_j)} $$

解析续： 求 $P(A_1|B)$。

  • 分子（甲导致次品的概率）：$P(A_1)P(B|A_1) = 0.3 \times 0.02 = 0.006$
  • 分母（全概率，刚刚算过）：$P(B) = 0.017$
  • 最终结果：$P(A_1|B) = \frac{0.006}{0.017} \approx \textcolor{#2980b9}{35.3\%}$

Session 4: 事件的独立性

Session 4: 独立性的定义

引入： 如果事件 A 的发生对事件 B 发生的概率毫无影响，我们就说它们是独立的。

相互独立的数学定义
设 A, B 是两个事件，如果满足： $$ P(AB) = P(A)P(B) $$ 则称事件 A 与事件 B 相互独立。

重要定理： 若 A 与 B 独立，则下列三对事件也相互独立：

A 与 $\bar{B}$   |   $\bar{A}$ 与 B   |   $\bar{A}$ 与 $\bar{B}$

Session 4: 防坑指南 (独立 vs 互斥)

互不相容 (互斥)
  • 表达式： $A \cap B = \emptyset$
  • 概率： $P(AB) = 0$
  • 含义： 你发生，我就绝对不发生。（属于强烈的排斥关系）
相互独立
  • 表达式： $P(AB) = P(A)P(B)$
  • 概率： 乘积关系
  • 含义： 你发不发生，跟我毫无关系。（属于完全的路人关系）

绝对真理： 除非其中一个是概率为0的不可能事件，否则互斥必然不独立，独立必然不互斥！

Session 4: 多个事件的独立性

三个事件 A, B, C 相互独立，要求比两两独立更严格！必须同时满足四个等式：

  • $P(AB) = P(A)P(B)$
  • $P(AC) = P(A)P(C)$
  • $P(BC) = P(B)P(C)$
  • $P(ABC) = P(A)P(B)P(C)$   $\leftarrow$ （缺一不可！）

工程意义： 多个独立元件组成的系统，联合概率可以直接连乘。这大大简化了复杂系统的可靠性计算。

Session 4: 经典例题 (至少发生问题)

例题： 甲、乙两人独立对同一目标进行射击，甲的命中率为 0.8，乙的命中率为 0.7。求目标被命中的概率。

解析 (正面硬刚法)： 分三种情况。
$P = P(A\bar{B}) + P(\bar{A}B) + P(AB) = 0.8\times0.3 + 0.2\times0.7 + 0.8\times0.7 = 0.94$。

解析 (逆向思维法 / 正难则反)：

目标被命中 = 至少一人命中。
对立事件 = 两人都没命中（即 $\bar{A}\bar{B}$）。

$$ P = 1 - P(\bar{A}\bar{B}) $$

因为独立，

$$ P = 1 - P(\bar{A})P(\bar{B}) = 1 - (1-0.8) \times (1-0.7) = 1 - 0.06 = \textcolor{#e74c3c}{0.94} $$

Session 4: 工程应用 (系统可靠性计算)

假设三个元件 1, 2, 3 独立工作，可靠性均为 $p$。

1. 串联系统 (一损俱损)：

  • 系统正常 $\Leftrightarrow$ 所有元件正常。
  • $P_{\text{串}} = P(A_1 A_2 A_3) = P(A_1)P(A_2)P(A_3) = \textcolor{#16a085}{p^3}$

2. 并联系统 (安全冗余)：

  • 系统正常 $\Leftrightarrow$ 至少一个元件正常。（正难则反！）
  • 系统损坏 $\Leftrightarrow$ 所有元件全部损坏。
  • $P_{\text{并}} = 1 - P(\bar{A_1}\bar{A_2}\bar{A_3}) = 1 - (1-p)^3$

随堂小测：概念与计算考核

选择题挑战

1. 若事件 A 和 B 满足 $P(A) > 0, P(B) > 0$，且 A 与 B 互不相容，则 A 与 B（ ）。

  1. 必然相互独立
  2. 必然不相互独立
  3. 不一定独立
  4. 是对立事件
👉 答案：B
解析：由于互斥，$P(AB)=0$。但 $P(A) > 0$ 且 $P(B) > 0$，所以 $P(A)P(B) > 0$。显然 $P(AB) \neq P(A)P(B)$，因此必然不独立。

2. 已知 $P(A)=0.4, P(B)=0.5$，且 A 与 B 相互独立，则 $P(A \cup B)$ 等于（ ）。

  1. 0.9
  2. 0.7
  3. 0.2
  4. 0.1
👉 答案：B
解析：因为独立，$P(AB) = P(A)P(B) = 0.4 \times 0.5 = 0.2$。再由加法公式 $P(A \cup B) = 0.4 + 0.5 - 0.2 = 0.7$。

概念与计算考核

考核 1： 在贝叶斯公式中，我们称 $P(A_i)$ 为先验概率。那么通过贝叶斯公式计算得出的结果 $P(A_i|B)$ 通常被称作什么概率？

👉 答案：后验概率

考核 2： 已知患某病的概率为 1%，某试剂检测准确率为 90%。若某人测出阳性，求真正患病的概率大约是多少？（只列公式即可）

👉 解析：（贝叶斯公式应用）

  • 设 A = "患病", $\bar{A}$ = "无病"。 B = "检测为阳性"。
  • $P(A)=0.01$, $P(\bar{A})=0.99$
  • $P(B|A)=0.9$, $P(B|\bar{A})=0.1$ （无病却误诊的概率）
  • $$ P(A|B) = \frac{0.01 \times 0.9}{0.01 \times 0.9 + 0.99 \times 0.1} \approx 8.3\% $$
