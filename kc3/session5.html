Session 5: 随机变量与离散型分布

Session 5: 为什么要引入随机变量？ (从事件到数字)

知识回顾： 在第一章里，我们习惯用 A, B, C 等字母来代表“正面朝上”、“次品”等具体事件。但这在进行数学运算时极不方便！

随机变量 (Random Variable) 的核心定义
假定某个随机试验的样本空间是 S。若对任意一个样本点 $\omega\in S$，都存在唯一确定的实数值 $X(\omega)$ 和它对应，则该实值函数 $X=X(\omega)$ 被统称为随机变量。

核心思想： 把抽象的“实验结果”转化为了具体的“数字”（即数字化）。

Session 5: 离散型随机变量与分布律

概念： 若某随机变量 X 所能取到的所有值表现为有限个，或者是可列举的无限多个，我们就叫它离散型随机变量。

分布律 (Probability Mass Function)
为了完整刻画离散型随机变量的特征，必须搞清楚两件事：它可能取到哪些数值？取得各个数值的对应概率又是多大？ $$ P(X=x_{k})=p_{k}, \quad k=1,2,3,... $$

分布律的表格呈现方式：

$X$ $x_1$ $x_2$ $...$ $x_k$ $...$
$p_k$ $p_1$ $p_2$ $...$ $p_k$ $...$

两大核心性质： (1) 非负性: $p_{k}\ge0$; (2) 规范性: $\sum_{k}p_{k}=1$.

Session 5: 常见离散分布一 —— (0-1) 分布

此为最基础的随机变量形式，常用于表示单次试验中“非黑即白”或“成败与否”的结果。

两点分布 / (0-1) 分布
假如随机变量 X 仅仅取 0 与 1 这两个数字，其概率分布满足： $$ P(X=1)=p, \quad P(X=0)=1-p=q \quad (0 \lt p \lt 1) $$ 则称 X 服从参数为 p 的 (0-1) 分布，记作 $X\sim B(1,p)$。

常见应用场景：

  • 质检一件产品：数字 1 代表残次品，0 代表合格品。
  • 进行一次投篮：数字 1 代表命中，0 代表打铁。

Session 5: 常见离散分布二 —— 二项分布

伯努利试验模型： 将单次成功概率为 p 的 (0-1) 试验，在完全相同的条件下独立地重复进行 n 次。

二项分布 (Binomial Distribution)
假设随机变量 X 代表了 n 重伯努利试验里“取得成功的总次数”。那么 X 可能取到的结果就是 $0,1,2,...,n$，对应的分布律为： $$ P(X=k)=C_{n}^{k}p^{k}(1-p)^{n-k}, \quad k=0,1,2,...,n $$ 称 X 服从参数为 n 和 p 的二项分布，记作 $X\sim B(n,p)$。

Session 5: "Sports Teacher” 专属例题 (二项分布)

例题： 某篮球运动员的罚球命中率保持在 0.8。在比赛的最后关键时刻，他获得了 3 次罚球机会。假设每次罚球相互独立。

求：(1) 他恰好罚进 2 球的概率；(2) 他至少罚进 1 球的概率。

解析： 这是一个典型的 $n=3$ 重伯努利试验。设 X 为罚进的球数，则 $X\sim B(3,0.8)$。

  • (1) 恰好进 2 球 $(X=2)$:
    $$ P(X=2)=C_{3}^{2}(0.8)^{2}(1-0.8)^{3-2}=3\times0.64\times0.2=\textcolor{#16a085}{0.384} $$
  • (2) 至少进 1 球 $(X\ge1)$：(利用正难则反!)
    $$ P(X\ge1)=1-P(X=0) $$ $$ P(X\ge1)=1-C_{3}^{0}(0.8)^{0}(0.2)^{3}=1-0.008=\textcolor{#e74c3c}{0.992} $$

Session 5: 常见离散分布三 —— 泊松分布

当二项分布中，试验次数 n 非常大，而单次成功率 p 非常小，且 $np=\lambda$(常数)时，二项分布的极限就是泊松分布。

泊松分布 (Poisson Distribution)
若随机变量 X 的所有可能取值为 $0,1,2,...$，且分布律为： $$ P(X=k)=\frac{\lambda^{k}e^{-\lambda}}{k!}, \quad k=0,1,2,... \quad (\lambda>0) $$ 则称 X 服从参数为 $\lambda$ 的泊松分布，记作 $X\sim P(\lambda)$ 或 $X\sim\pi(\lambda)$。

现实意义(描述“稀有事件”的发生次数):

  • 一小时内到达某十字路口的车辆数。
  • 一本书中每一页印刷错误的字数。
  • 某放射性物质在规定时间内放出的粒子数。

随堂小测：概念与计算考核

选择题挑战

1. 设离散型随机变量 X 的所有可能取值为 -1, 0, 1，对应的概率分别为 0.2, 0.5, p，则 p 的值必然是（ ）。

  1. 0.1
  2. 0.2
  3. 0.3
  4. 0.5
👉 答案：C
解析：考查分布律的规范性（概率和为 1）。$0.2 + 0.5 + p = 1$，解得 $p = 0.3$。

2. 下列哪种随机现象最适合用“泊松分布”来描述？（ ）

  1. 抛掷 10 枚硬币，观察正面朝上的总个数
  2. 某篮球老师连续罚球 5 次，罚进的次数
  3. 检查一件工厂流水线上的产品是否为次品
  4. 某偏僻的十字路口在一小时内经过的车辆数
👉 答案：D
解析：泊松分布常用于描述单位时间/空间内“稀有事件”发生的次数。 选项 A、B 属于二项分布，选项 C 属于 (0-1) 分布。

概念与计算考核

考核 1： 若某随机变量 $X \sim B(1, p)$，说明 X 服从什么分布？它通常用来描述什么类型的试验？

👉 答案： 服从 (0-1) 分布（或两点分布）。通常用来描述只有两种可能结果（“非黑即白”、“成功或失败”）的单次试验。

考核 2： 某射击运动员每次击中目标的概率为 0.9。现他独立进行了 3 次射击，设击中目标的总次数为 X。请写出 X 服从的分布，以及“三次全中”的概率算式。

👉 解析：

  • 这是一个 n=3，p=0.9 的独立重复（伯努利）试验。
  • 因此 X 服从二项分布，记作：$X \sim B(3, 0.9)$
  • 三次全中即要求 $X=3$。代入二项分布公式：
    $$ P(X=3) = C_3^3 (0.9)^3 (1-0.9)^0 = 0.9^3 = 0.729 $$
