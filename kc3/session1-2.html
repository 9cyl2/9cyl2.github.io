Session 1: 绪论 & 随机事件

Session 1: 课程简介与导语

什么是概率论与数理统计？

  • 确定性现象： 在一定条件下，必然发生某种结果的现象。（如：向上抛石头，必然下落；水加热到100度必然沸腾）。
  • 随机现象： 在一定条件下，可能出现这种结果，也可能出现那种结果，事先无法预知。（如：抛掷一枚硬币；明天是否下雨；股票的涨跌）。
学科使命
概率论与数理统计就是一门研究和探索客观世界随机现象规律性的数学学科。

Session 1: 随机试验 (Random Experiment)

为了研究随机现象，我们需要对其进行观察或实验。把满足以下三个条件的试验称为随机试验 ($E$)：

  1. 可重复性： 可以在相同的条件下重复进行；
  2. 可知性： 每次试验的可能结果不止一个，且能事先明确所有可能结果；
  3. 不可预知性： 进行一次试验之前不能确定哪一个结果会出现。

互动思考： “掷一颗骰子，观察出现的点数” 是否为随机试验？

Session 1: 样本空间与样本点

  • 样本空间 ($S$ 或 $\Omega$)： 随机试验 $E$ 的所有可能结果组成的集合。
  • 样本点 ($\omega$)： 样本空间的元素，即试验 $E$ 的每一个可能结果。

例题演示：写出下列随机试验的样本空间

  1. $E_1$：抛掷一枚硬币，观察正面 (H) 与反面 (T) 出现的情况。
    $\Rightarrow S_1 = \{H, T\}$
  2. $E_2$：记录某城市一天的交通事故总数。
    $\Rightarrow S_2 = \{0, 1, 2, \dots \}$
  3. $E_3$：测试某批次灯泡的寿命 $t$（小时）。
    $\Rightarrow S_3 = \{t \,|\, t \ge 0\}$

Session 1: 随机事件 (Random Events)

定义： 样本空间 $S$ 的子集称为随机事件，简称事件。

  • 发生： 当且仅当这一子集中的一个样本点出现时，称这一事件发生。
  • 基本事件： 由一个样本点组成的单点集。
  • 必然事件 ($S$)： 每次试验中必定发生的事件。
  • 不可能事件 ($\emptyset$)： 每次试验中必定不发生的事件。

Session 1: 互不相容与对立事件 (核心考点)

互不相容 (互斥) 事件
  • $A \cap B = \emptyset$
  • 含义：$A$ 与 $B$ 不能同时发生。
对立事件 (逆事件 $\bar{A}$)
  • $A \cup B = S$ 且 $A \cap B = \emptyset$
  • 含义：每次试验中 $A, B$ 仅有一个发生。$B = \bar{A}$。

易错点提示： 对立必然互斥，互斥不一定对立！


Session 1: 课堂例题解析

例题： 设 $A, B, C$ 为三个事件，用运算关系式表示下列事件：

1. $A, B, C$ 都不发生：

$\bar{A}\bar{B}\bar{C}$ 或 $\overline{A \cup B \cup C}$

2. $A, B, C$ 中至少有一个发生：

$A \cup B \cup C$

3. $A, B, C$ 中恰有一个发生：

$A\bar{B}\bar{C} \cup \bar{A}B\bar{C} \cup \bar{A}\bar{B}C$

Session 2: 随机事件的概率 & 古典概率

Session 2: 频率与概率的演进

  • 频率 $f_n(A)$： 在 $n$ 次试验中，事件 $A$ 发生的次数 $n_A$ 的比值 $f_n(A) = \frac{n_A}{n}$。
  • 频率的稳定性： 试验次数 $n$ 极大时，频率会在一个稳定值 $p$ 附近摆动。
  • 概率的统计定义： 这个稳定的客观数值 $p$ 就是概率 $P(A)$。
公理化定义 (柯尔莫哥洛夫，1933)
为了数学上的严谨，规定概率必须满足：非负性、规范性 ($P(S)=1$)、可列可加性。

Session 2: 概率的加法公式 (极其重要)

加法公式
对于任意两个事件 $A$ 和 $B$：
$$ P(A \cup B) = P(A) + P(B) - P(AB) $$

推广到三个事件：

$$ P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC) $$

直观理解： 韦恩图面积的计算（容斥原理）。两圆相交多加了的交集面积要减去一次。


Session 2: 古典概型 (等可能概型)

定义： 若随机试验满足以下两个条件，则称为古典概型：

  1. 有限性： 样本空间 $S$ 只包含有限个样本点 $n$。
  2. 等可能性： 每个基本事件发生的概率相等。
古典概型计算公式
设事件 $A$ 包含 $m$ 个样本点，则： $$ P(A) = \frac{m}{n} = \frac{\text{事件 } A \text{ 包含的样本点数}}{\text{样本空间 } S \text{ 的总样本点数}} $$

Session 2: 古典概型例题 —— 摸球模型

例题： 袋中有 3 个红球和 4 个白球。从中无放回地任取 2 个球。
求：(1) 两个都是红球的概率； (2) 一红一白的概率。

解析：

  • 样本空间总点数（从 7 个球中取 2 个）：$n = C_7^2 = 21$。
  • (1) 设 $A$ = "全是红球"：
    取法数为 $m_1 = C_3^2 = 3$。
    $P(A) = \frac{3}{21} = \frac{1}{7}$。
  • (2) 设 $B$ = "一红一白"：
    利用乘法原理，取法数为 $m_2 = C_3^1 \times C_4^1 = 12$。
    $P(B) = \frac{12}{21} = \frac{4}{7}$。
