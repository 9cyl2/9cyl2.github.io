Session 1: 课程简介与导语

什么是概率论与数理统计？

  • 确定性现象： 在一定条件下，必然发生某种结果的现象。（如：向上抛石头，必然下落；水加热到100度必然沸腾）。
  • 随机现象： 在一定条件下，可能出现这种结果，也可能出现那种结果，事先无法预知。（如：抛掷一枚硬币；明天是否下雨；股票的涨跌）。
学科使命
概率论与数理统计就是一门研究和探索客观世界随机现象规律性的数学学科。

Session 1: 随机试验与样本空间

满足 可重复性、可知性、不可预知性 三个条件的试验称为随机试验 ($E$)。

  • 样本空间 ($S$)： 随机试验 $E$ 的所有可能结果组成的集合。
  • 样本点 ($\omega$)： 样本空间的元素，即试验的每一个可能结果。
书本经典例题：写出样本空间

例： 抛掷两枚硬币，观察正面 (H) 与反面 (T) 出现的情况。

$\Rightarrow S = \{ (H,H), (H,T), (T,H), (T,T) \}$

例： 记录某城市一天的交通事故总数。

$\Rightarrow S = \{ 0, 1, 2, \dots \}$

Session 1: 事件的关系与运算

互不相容 (互斥) 事件
  • $A \cap B = \emptyset$
  • 含义：$A$ 与 $B$ 不能同时发生。
对立事件 (逆事件 $\bar{A}$)
  • $A \cup B = S$ 且 $A \cap B = \emptyset$
  • 含义：每次试验中 $A, B$ 必有一个发生。

提示： 对立必然互斥，互斥不一定对立！

Session 1: 书本典型例题（产品抽样）

例题： 某工厂生产了3件产品，以 $A_i$ 表示“第 $i$ 件是正品” ($i=1,2,3$)。试用 $A_i$ 的运算关系式表示下列事件：

(1) 只有第1件是正品：

$A_1 \bar{A_2} \bar{A_3}$

(2) 至少有一件是次品（即不全为正品）：

$\bar{A_1} \cup \bar{A_2} \cup \bar{A_3}$ 或 $\overline{A_1 A_2 A_3}$ (利用德·摩根律)

(3) 恰好有一件是正品：

$A_1 \bar{A_2} \bar{A_3} \cup \bar{A_1} A_2 \bar{A_3} \cup \bar{A_1} \bar{A_2} A_3$

Session 2: 随机事件的概率 & 古典概率

Session 2: 概率的性质与加法公式

  • 规范性： $P(S) = 1, P(\emptyset) = 0$
  • 对立事件： $P(\bar{A}) = 1 - P(A)$
加法公式 (重点)
对于任意两个事件 $A$ 和 $B$：
$$ P(A \cup B) = P(A) + P(B) - P(AB) $$

直观理解： 韦恩图面积的计算（容斥原理）。多加的交集部分要减去一次。

Session 2: 古典概型 (等可能概型)

定义： 若随机试验满足 有限性 (样本点有限) 和 等可能性 两个条件，则称为古典概型。

古典概型计算公式
设事件 $A$ 包含 $m$ 个样本点，样本空间总点数为 $n$，则： $$ P(A) = \frac{m}{n} = \frac{\text{事件 } A \text{ 包含的样本点数}}{\text{样本空间 } S \text{ 的总样本点数}} $$

核心解题工具： 排列 ($P_n^k$ 或 $A_n^k$) 与 组合 ($C_n^k$)

Session 2: 书本经典例题 (双骰子模型)

例题： 抛掷两颗均匀的骰子，求出现的点数之和为 8 的概率。

解析：

  • 每一颗骰子有6种可能，由乘法原理，样本空间总点数为：$n = 6 \times 6 = 36$
  • 设事件 $A$ = "点数之和为8"。
    列举 $A$ 包含的样本点：$(2,6), (3,5), (4,4), (5,3), (6,2)$。
    共包含 $m = 5$ 个样本点。
  • 故概率为： $$ P(A) = \frac{m}{n} = \frac{5}{36} $$

Session 2: 书本经典例题 (次品抽样模型)

例题： 一批产品共有 10 件，其中有 3 件次品，7 件正品。从中任取 2 件，求：

(1) 恰好取到 1 件次品的概率；
(2) 至少取到 1 件次品的概率。

解析： 样本空间总点数 $n = C_{10}^2 = 45$。

  • (1) 恰有 1 件次品 ($A$)：
    从 3 件次品中取 1 件，7 件正品中取 1 件。
    $P(A) = \frac{C_3^1 \times C_7^1}{C_{10}^2} = \frac{3 \times 7}{45} = \frac{7}{15}$
  • (2) 至少 1 件次品 ($B$) (正难则反)：
    对立事件 $\bar{B}$ = "全是正品"。
    $P(B) = 1 - P(\bar{B}) = 1 - \frac{C_7^2}{C_{10}^2} = 1 - \frac{21}{45} = \frac{8}{15}$

选择题挑战

1. 若事件 A 和 B 满足 $A \cup B = S$ 且 $A \cap B = \emptyset$，则称 A 与 B 为（ ）。

  1. 互不相容事件
  2. 对立事件
  3. 独立事件
  4. 必然事件
👉 答案：B
解析：这是对立事件的严格定义。对立必然互斥，但仅仅互斥（不相交）不一定能拼成完整的样本空间 $S$。

2. 已知 $P(A)=0.5, P(B)=0.4, P(A \cup B)=0.7$，则 $P(AB)$ 等于（ ）。

  1. 0.1
  2. 0.2
  3. 0.3
  4. 0.9
👉 答案：B
解析：考查加法公式。$P(A \cup B) = P(A) + P(B) - P(AB)$，代入得 $0.7 = 0.5 + 0.4 - P(AB)$，解得 $P(AB) = 0.2$。

概念与计算考核

考核 1： 设 $A, B, C$ 为三个事件，请用运算关系式表示事件“$A, B, C$ 中恰有两个发生”。

👉 答案： $AB\bar{C} \cup A\bar{B}C \cup \bar{A}BC$

考核 2： 抛掷一枚均匀的硬币 3 次，求“至少出现一次正面”的概率。

👉 解析：（提示：正难则反）

  • 样本空间总点数 $n = 2 \times 2 \times 2 = 8$。
  • 设事件 $A$ = "至少出现一次正面"，则对立事件 $\bar{A}$ = "三次全为反面"。
  • $\bar{A}$ 包含的样本点只有 1 个（即 {反,反,反}）。
  • 最终概率：$P(A) = 1 - P(\bar{A}) = 1 - \frac{1}{8} = \frac{7}{8}$
