Session 11: 协方差与相关系数

Session 11: 为什么需要协方差？

引入： 方差 $D(X)$ 只能衡量“单个变量”的波动。但在体育数据中，我们经常想知道两个变量是如何协同波动的。

  • 比如：球员的身高 X 与 场均篮板数 Y。通常身高越高，篮板越多（同向波动）。
  • 比如：百米赛跑的训练时长 X 与 冲刺时间 Y。训练越久，时间越短（反向波动）。
协方差 (Covariance) 的定义
为了衡量这种协同波动的关系，我们定义 X 与 Y 的协方差为： $$ Cov(X, Y) = E \{ [X - E(X)][Y - E(Y)] \} $$

核心计算公式（考试就用它！）：
$ Cov(X, Y) = E(XY) - E(X)E(Y) $

Session 11: 协方差的性质

协方差有几个非常重要的代数性质，大家可以把它想象成向量的“内积”。

协方差的核心性质
  • 1. 对称性： $Cov(X, Y) = Cov(Y, X)$
  • 2. 自身即方差： $Cov(X, X) = D(X)$ （自己和自己的协同波动，就是自身的方差）
  • 3. 线性提出常数： $Cov(aX, bY) = ab \cdot Cov(X, Y)$
  • 4. 分配律： $Cov(X+Y, Z) = Cov(X, Z) + Cov(Y, Z)$

方差展开公式（上节课挖的坑今天补上了）：
$$ D(X \pm Y) = D(X) + D(Y) \pm 2Cov(X, Y) $$ （只有当 X 与 Y 独立时，$Cov(X,Y)=0$，后一项才会消失！）

Session 11: 相关系数 (Correlation Coefficient)

协方差的缺陷： 它的数值大小受单位影响极大！比如用米和厘米算出的协方差完全不同，导致难以统一评判相关性的“强弱”。

相关系数的定义 (标准化)
为了消除量纲（单位）的影响，我们将协方差除以两者的标准差，得到相关系数 $\rho_{XY}$： $$ \rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)} \sqrt{D(Y)}} $$

神奇的边界性质：
无论 X 和 Y 是什么分布，以下不等式永远成立！
$ -1 \le \rho_{XY} \le 1 $

Session 11: 相关系数的几何直观 (散点图)

相关系数 $\rho$ 本质上衡量的是两个变量之间的线性相关程度（即散点图有多像一条直线）。

极端情况： 当 $|\rho| = 1$ 时，散点百分之百落在一条直线上，说明 X 和 Y 存在严格的线性函数关系 $Y = aX + b$。

Session 11: 独立 vs 不相关 (期末选择题必考坑点！)

“独立” 和 “不相关” 是一回事吗？绝对不是！

两者的逻辑关系
  • 独立 $\Rightarrow$ 不相关： 如果两个变量互相独立（完全没关系），那么它们肯定没有“线性关系”，所以协方差为0，必然不相关。
  • 不相关 $\not\Rightarrow$ 独立： 不相关仅仅代表没有“线性关系”！它们可能存在强烈的“曲线关系”（比如 $Y = X^2$），此时算出来 $\rho = 0$，但它们显然是不独立的。

唯一的例外（超级重点）：
当 $(X,Y)$ 服从二维正态分布时，独立 和 不相关 是完全等价的！

Session 11: "Sports Teacher” 专属例题 (协方差计算)

例题： 统计某控卫的单场助攻数 X (5或10) 与得分 Y (10或20) 的联合分布律如下表。求 $Cov(X,Y)$。

$X \setminus Y$$10$ 分$20$ 分$P(X)$
$5$ 次$0.4$$0.1$$0.5$
$10$ 次$0.1$$0.4$$0.5$
$P(Y)$$0.5$$0.5$$1.0$

解析： 使用公式 $Cov(X,Y) = E(XY) - E(X)E(Y)$。

  • 算边缘期望：$E(X) = 5(0.5)+10(0.5) = 7.5$； $E(Y) = 10(0.5)+20(0.5) = 15$。
  • 算联合期望：$E(XY) = \sum\sum (x_i y_j \cdot p_{ij}) $
    $= (5\times10\times0.4) + (5\times20\times0.1) + (10\times10\times0.1) + (10\times20\times0.4) = 120$。
  • $Cov(X,Y) = 120 - 7.5 \times 15 = 120 - 112.5 = \textcolor{#e74c3c}{7.5}$。
  • 结论： 协方差 $\gt 0$，说明助攻与得分正相关（状态好时两者往往同时高）。

随堂小测：概念与计算考核

选择题挑战

1. 已知随机变量 $X$ 和 $Y$ 满足 $D(X+Y) = D(X) + D(Y)$，则下列说法一定正确的是（ ）。

  1. $X$ 和 $Y$ 相互独立
  2. $X$ 和 $Y$ 不相关
  3. $X$ 和 $Y$ 呈完全正相关
  4. $X$ 和 $Y$ 服从正态分布
👉 答案：B
解析：由方差展开公式 $D(X+Y) = D(X) + D(Y) + 2Cov(X,Y)$ 可知，如果该等式成立，则说明 $Cov(X,Y)=0$。协方差为0的定义就是两者“不相关”。注意，不能选A，因为不相关推不出独立。

2. 若随机变量 $X, Y$ 的相关系数 $\rho_{XY} = -1$，则说明它们之间存在（ ）。

  1. 严格的正比例函数关系
  2. 严格的反比例（负相关）线性关系
  3. 没有任何关系
  4. 相互独立
👉 答案：B
解析：当 $|\rho| = 1$ 时，两变量存在严格的线性关系 $Y = aX+b$。又因为 $\rho = -1 < 0$，所以系数 $a < 0$，即存在严格的递减线性关系。

概念与计算考核

考核 1：概念辨析 “如果 X 和 Y 相互独立，那么它们一定不相关。反过来，如果 X 和 Y 不相关，它们也一定相互独立。”这句话对吗？为什么？

👉 答案：前半句对，后半句错。 独立一定能推出不相关；但不相关（$\rho=0$）只能说明没有“线性关系”，它们可能存在其他非线性的函数关系（如 $Y=X^2$），此时显然不独立。唯有在二维正态分布中，两者才完全等价。

考核 2：综合计算 已知 $D(X) = 4, D(Y) = 9$，且 $X$ 和 $Y$ 的相关系数 $\rho_{XY} = 0.5$。求 $Cov(X,Y)$ 以及 $D(X - Y)$ 的值。

👉 解析：

  • 由公式 $\rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)} \sqrt{D(Y)}}$，代入数据得 $0.5 = \frac{Cov(X,Y)}{\sqrt{4}\sqrt{9}} = \frac{Cov(X,Y)}{2 \times 3}$。
    得出 $Cov(X,Y) = 0.5 \times 6 = 3$。
  • 利用方差的加减公式：
    $D(X - Y) = D(X) + D(Y) - 2Cov(X,Y)$
    $= 4 + 9 - 2(3) = 13 - 6 = 7$。
