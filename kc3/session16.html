Session 16: 假设检验

统计学的终极法庭：用数据拆穿谎言

Session 16: 什么是假设检验？

引入： 某运动品牌推销一款“革命性”短跑钉鞋，号称能显著提升百米成绩。我们校队以前的平均成绩是 12.0 秒。现在让 16 名队员穿上新鞋跑，测得平均成绩是 11.5 秒。

教练的灵魂拷问
平均成绩提升了 0.5 秒，到底是因为新鞋真的有用，还是仅仅因为今天这 16 个人吃得好状态佳（纯粹由于抽样带来的随机运气）？

假设检验的逻辑：反证法 + 小概率原理
我们先“假定”新鞋没用（大家还是平时的水平），然后看看在这种假定下，出现 11.5 秒这种极端好成绩的概率有多大。如果算出来概率极小（比如不到 5%），我们就强行推翻最初的假定，承认新鞋真的牛逼！

Session 16: 原假设与备择假设

我们在做检验前，必须先确立场上的“控辩双方”。

提出假设 (Hypotheses)
  • 原假设 (Null Hypothesis, $H_0$)： 通常代表“没有变化”、“没有效果”、“等于历史常态值”。它是我们想要去推翻的靶子。
    （比如：$H_0: \mu = 12.0$，新鞋其实没用）
  • 备择假设 (Alternative Hypothesis, $H_1$)： 通常代表“发生变化了”、“有显著提升”。它是我们真正想要去证明的结论。
    （比如：$H_1: \mu \lt 12.0$，所需时间真的变短了）

核心铁律：统计学的法庭默认 $H_0$ 是无罪的。除非你能提供“铁证”（足够极端的样本数据），否则绝不能轻易拒绝 $H_0$！

Session 16: 两类错误 (Two Types of Errors)

既然是依据局部概率做整体判决，裁判（统计学家）就一定会发生误判：

真实情况 $\backslash$ 我们的判决 拒绝 $H_0$
(判定新鞋有用)		 接受 $H_0$
(判定新鞋没用)
$H_0$ 实际上为真
(新鞋是个智商税)		 第一类错误 (弃真)
(冤枉好人 / 假阳性)		 正确判决 😊
$H_0$ 实际上为假
(新鞋真的能提速)		 正确判决 😊 第二类错误 (存伪)
(放走坏人 / 假阴性)

显著性水平 $\alpha$： 即犯第一类错误的最大允许概率（通常设为 0.05 或 0.01）。含义是：“我最多只允许自己有 5% 的概率把无效的鞋吹成神鞋”。

Session 16: 假设检验标准四步曲

期末考试面对任何假设检验的大题，只要严格按照这四步走，一定能拿满分！

解题模板四步法
  1. 提假设： 根据题意写出 $H_0$ 和 $H_1$。（看是双侧、左侧还是右侧检验）
  2. 找统计量： 确定检验统计量（总体方差已知用 $Z$，方差未知用 $t$），并写出它在 $H_0$ 成立条件下的理论分布。
  3. 定拒绝域： 根据显著性水平 $\alpha$ 和检验方向，查表找出临界值，画出拒绝域 (Rejection Region, W)。
  4. 做判决： 把实际的样本数据代入公式，算出一个检验的实测值。如果它落入拒绝域 $W$ 中，就强力拒绝 $H_0$；否则不拒绝 $H_0$。

Session 16: "Sports Teacher” 专属例题 (单侧 t 检验)

接着上面引入的例子： 过去队里百米均值 $\mu_0 = 12.0$ 秒。穿新鞋抽测 $n=16$ 人，样本均值 $\bar{x} = 11.5$ 秒，样本标准差 $s = 0.8$ 秒。在显著性水平 $\alpha=0.05$ 下，新鞋是否真有提速效果？

解析： 总体方差未知，小样本($n=16$)，必须用 t 检验。

  • 1. 提假设： $H_0: \mu \ge 12.0$ (没提速)； $H_1: \mu \lt 12.0$ (提速了，左侧检验)
  • 2. 统计量： 假定 $H_0$ 成立，则 $t = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \sim t(n-1) = t(15)$。
  • 3. 拒绝域： 左侧检验找左边尾巴。查表知 $t_{0.05}(15) = 1.753$。所以拒绝域为 $W = \{ t \lt -1.753 \}$。
  • 4. 算实测值： $t_{obs} = \frac{11.5 - 12.0}{0.8 / \sqrt{16}} = \frac{-0.5}{0.2} = \textcolor{#c0392b}{-2.5}$。
  • 5. 结论： 因为 $-2.5 \lt -1.753$，落入拒绝域！所以果断拒绝 $H_0$，承认新鞋确实能显著提速！

随堂小测：概念与最终考核

选择题挑战

1. 在假设检验中，所谓的“显著性水平 $\alpha$” 代表的是（ ）。

  1. 原假设 $H_0$ 为真的概率
  2. 犯第二类错误（存伪）的概率
  3. 当 $H_0$ 为真时，却错误地拒绝了 $H_0$ 的概率
  4. 检验统计量落入接受域的概率
👉 答案：C
解析：$\alpha$ 就是犯第一类错误（弃真错误 / 冤枉好人）的最大概率限制。

2. 当我们在做假设检验时，如果我们非常不想“冤枉好人”（即极度害怕犯第一类错误），我们应该把显著性水平 $\alpha$ （ ）。

  1. 调大（比如从 0.05 调到 0.10）
  2. 调小（比如从 0.05 调到 0.01）
  3. 保持不变
  4. 增加样本容量
👉 答案：B
解析：$\alpha$ 是犯第一类错误的概率上限。调小 $\alpha$ 可以直接降低冤枉好人的概率（让拒绝域变得极小，除非证据极其确凿否则不拒绝）。但这会导致第二类错误（放走坏人）的概率急剧增加。

概念与逻辑考核

考核 1：概念辨析 如果经过计算，实测统计量落在了“接受域”，我们最终的结论术语应该写“证明了原假设 $H_0$ 是完全正确的”，还是写“没有足够的证据拒绝 $H_0$”？为什么？

👉 答案： 必须写“没有足够的证据拒绝 $H_0$”（或“接受 $H_0$”）。在统计学中，我们永远无法用样本“100% 证明”一个总体假设是真的。我们只能说，目前的样本数据看起来不算太离谱，不足以让我们推翻它。这和法庭上判决“证据不足，疑罪从无”是一个道理，不代表被告绝对是个圣人。

考核 2：流程默写 回忆一下，对一个方差已知的大样本总体均值 $\mu$ 进行双侧检验（看是否有显著差异）时，检验统计量该写成什么公式？如果显著性水平是 $\alpha$，拒绝域的判断条件是什么？

👉 解析：

  • 方差 $\sigma^2$ 已知，必须使用 Z 检验。
  • 检验统计量公式：$ Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}} $。
  • 因为是双侧检验，所以要把 $\alpha$ 分摊到两边尾巴（各自 $\alpha/2$）。拒绝域条件为：$|Z_{obs}| \ge z_{\alpha/2}$。如果算出来的绝对值比临界值还大，就拒绝原假设。
