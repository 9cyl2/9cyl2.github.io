Session 13: 数理统计的基本概念与抽样分布

Session 13: 从概率论走向数理统计

引入： 前半学期我们学的“概率论”是“从因推果”（已知这批篮球次品率为10%，求抽到次品的概率）。但在现实采购中，厂家绝不会告诉你次品率！

数理统计的核心逻辑：从果推因
我们只能通过实际抽样测试（比如随机拍了 20 个篮球，发现 2 个漏气），利用这有限的数据样本，去反向推测和估计整批篮球的真实质量。

Session 13: 统计量 (Statistic) —— 浓缩就是精华

拿到 20 个篮球的弹跳高度数据后，直接看一堆数字会很晕，我们需要把它们浓缩成几个关键的指标，这些加工后的指标就叫统计量。

最核心的两个统计量 (期末必背！)
设 $X_1, X_2, \dots, X_n$ 为来自某个总体的样本数据：
  • 样本均值 (Sample Mean)： $\bar{X} = \frac{1}{n} \sum_{i=1}^{n} X_i$
  • 样本方差 (Sample Variance)： $S^2 = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar{X})^2$

防坑预警（无数人死在这里）：
为什么样本方差除以的是 $n-1$ 而不是 n？
因为 $\bar{X}$ 已经提前消耗掉了一个“自由度”！
（体育大白话：篮球队 5 个人平均身高已知，只要告诉你前 4 个人的身高，第 5 个人的身高就被锁死了，所以能“自由变化”的数据实际上只有 4 个！）

Session 13: "Sports Teacher” 专属例题 (计算统计量)

例题： 体育器材室新进了一批铅球。老师为了测试其重量是否达标，随机抽查了 5 个铅球，测得重量（单位：kg）分别为：

$ 7.2, \quad 7.1, \quad 7.4, \quad 7.0, \quad 7.3 $

求这组测试样本的样本均值 $\bar{X}$ 和 样本方差 $S^2$。

解析：

  • 算样本均值：
    $\bar{X} = \frac{7.2 + 7.1 + 7.4 + 7.0 + 7.3}{5} = \frac{36.0}{5} = \textcolor{#2980b9}{7.2 \text{ kg}}$
  • 算样本方差（切记除以 $n-1 = 4$）：
    $S^2 = \frac{1}{4} [ (7.2-7.2)^2 + (7.1-7.2)^2 + (7.4-7.2)^2 + (7.0-7.2)^2 + (7.3-7.2)^2 ]$
    $S^2 = \frac{1}{4} [ 0 + 0.01 + 0.04 + 0.04 + 0.01 ] = \frac{0.10}{4} = \textcolor{#e74c3c}{0.025}$

Session 13: 抽样分布巨头一 —— $\chi^2$ (卡方) 分布

由于样本是随机抽的，算出来的 $\bar{X}$ 和 $S^2$ 也是随机的！它们服从什么规律呢？为此，统计学家发明了三大经典抽样分布。

1. $\chi^2$ (Chi-square) 分布
设 $Z_1, Z_2, \dots, Z_n$ 是相互独立且服从标准正态分布 $N(0,1)$ 的随机变量。
把它们的平方加起来： $$ \chi^2 = Z_1^2 + Z_2^2 + \dots + Z_n^2 $$ 就说这个和服从自由度为 n 的卡方分布，记作 $\chi^2(n)$。

重要性质： 期望 $E(\chi^2) = n$，方差 $D(\chi^2) = 2n$。
（提示：因为是平方相加，所以卡方变量的值永远大于等于0，且图像呈偏右的拖尾形态！）

Session 13: 巨头二与三 —— t 分布与 F 分布

2. Student's t 分布 (学生分布)
由一个标准正态分布 $X \sim N(0,1)$ 和一个卡方分布 $Y \sim \chi^2(n)$ 构成： $$ t = \frac{X}{\sqrt{Y/n}} \sim t(n) $$ 形态： 和正态分布一样关于 y 轴对称，但比正态分布“矮胖”（厚尾现象），专门用来对付小样本量 ($n \lt 30$) 的体育数据分析！
3. F 分布 (方差比分布)
由两个独立的卡方分布 $U \sim \chi^2(n_1)$ 和 $V \sim \chi^2(n_2)$ 构成比值： $$ F = \frac{U/n_1}{V/n_2} \sim F(n_1, n_2) $$ 应用： 体育界常用来比较两支球队的方差（发挥稳定性）是否有显著差异。

Session 13: 正态总体的样本分布定理 (极度重要)

在实际操作中，如果我们的总体（比如全校男生的身高）服从正态分布 $N(\mu, \sigma^2)$，那么抽样算出来的 $\bar{X}$ 和 $S^2$ 将满足以下两条神级定理：

  • 定理 1 (关于样本均值)： $\bar{X}$ 依然服从正态分布，且波动变小了！ $$ \bar{X} \sim N\left(\mu, \frac{\sigma^2}{n}\right) \quad \Rightarrow \quad \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1) $$
  • 定理 2 (关于样本方差)： $S^2$ 经过变换后服从卡方分布！ $$ \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1) $$

这两条定理是我们下一章做“参数估计”和“假设检验”的基础基石！

随堂小测：概念与计算考核

选择题挑战

1. 在计算样本方差 $S^2$ 时，为了保证其作为总体方差 $\sigma^2$ 的无偏估计，公式中除以的分母应当是（ ）。

  1. $n$
  2. $n-1$
  3. $n+1$
  4. $n-2$
👉 答案：B
解析：这是数理统计中最容易犯错的基础知识。因为使用样本均值 $\bar{X}$ 代替了未知的总体均值 $\mu$，导致丧失了一个自由度，所以必须除以 $n-1$ 才能成为无偏估计量。

2. 设 $X \sim N(0,1)$， $Y \sim \chi^2(n)$，且 X 与 Y 相互独立。则统计量 $T = \frac{X}{\sqrt{Y/n}}$ 服从的分布是（ ）。

  1. 标准正态分布 $N(0,1)$
  2. 卡方分布 $\chi^2(n)$
  3. 学生 t 分布 $t(n)$
  4. F 分布 $F(1, n)$
👉 答案：C
解析：这是 t 分布的严谨数学构造定义。“一个标准正态” 除以 “一个卡方除以其自由度的平方根”，就构成 t 分布。

概念与计算考核

考核 1：简答题 在统计学中，t 分布（Student's t-distribution）的概率密度曲线与标准正态分布曲线相比，在形态上有什么显著区别？它主要应用于什么场景？

👉 答案： t 分布的曲线比标准正态分布更矮、更胖（表现为“厚尾现象”或“长尾”），即在极值区域的概率比正态分布更高。它主要用于小样本（$n \lt 30$）情况下的统计推断。

考核 2：定理应用 设总体 $X$ 服从正态分布 $N(50, 16)$。现从中抽取一个容量为 $n=4$ 的样本，求样本均值 $\bar{X}$ 的期望和方差。

👉 解析：

  • 根据正态总体样本均值定理：$\bar{X} \sim N(\mu, \frac{\sigma^2}{n})$。
  • 已知总体期望 $\mu = 50$，总体方差 $\sigma^2 = 16$，样本容量 $n = 4$。
  • 所以 $\bar{X}$ 的期望 $E(\bar{X}) = \mu = \textcolor{#2980b9}{50}$。
  • $\bar{X}$ 的方差 $D(\bar{X}) = \frac{\sigma^2}{n} = \frac{16}{4} = \textcolor{#e74c3c}{4}$。
  • （这说明，样本均值的波动程度远远小于单个个体的波动程度，这符合“平均值更稳定”的直觉）。
